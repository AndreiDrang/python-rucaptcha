GeeTest¶

To import this module:

from python_rucaptcha.gee_test import GeeTest
class python_rucaptcha.gee_test.GeeTest(websiteURL: str, method: GeetestEnm | str, gt: str, version: int = 3, initParameters: dict = None, *args, **kwargs)¶
__init__(websiteURL: str, method: GeetestEnm | str, gt: str, version: int = 3, initParameters: dict = None, *args, **kwargs)¶

The class is used to work with GeeTest.

Parameters:

  • rucaptcha_key – User API key

  • websiteURL (str) – Full URL of the captcha page

  • gt (str) – The value of the gt parameter found on the site

  • version (int) – GeeTest V4 captcha should be set to 4.

  • initParameters (dict) – Required for GeeTest V4. The parameter that is passed in the initGeetest4 function call must contain the captcha_id value. Example of usage: { “captcha_id” : “e392e1d7fd421dc63325744d5a2b9c73”}

  • method (GeetestEnm | str) – Captcha type

  • kwargs – Not required params for task creation request
