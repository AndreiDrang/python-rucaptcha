Prosopo¶

To import this module:

from python_rucaptcha.prosopo import Prosopo
class python_rucaptcha.prosopo.Prosopo(websiteURL: str, websiteKey: str, method: str | ProsopoEnm = ProsopoEnm.ProsopoTaskProxyless, *args, **kwargs)¶
__init__(websiteURL: str, websiteKey: str, method: str | ProsopoEnm = ProsopoEnm.ProsopoTaskProxyless, *args, **kwargs)¶

The class is used to work with Prosopo.

Parameters:

  • rucaptcha_key – User API key

  • websiteURL (str) – The full URL of target web page where the captcha is loaded. We do not open the page, not a problem if it is available only for authenticated users

  • websiteKey (str) – The value of siteKey parameter found on the page.

  • method (str | ProsopoEnm) – Captcha type
