CaptchaFox¶

To import this module:

from python_rucaptcha.captcha_fox import CaptchaFox
class python_rucaptcha.captcha_fox.CaptchaFox(websiteURL: str, websiteKey: str, userAgent: str, proxyType: str, proxyAddress: str, proxyPort: str, *args, **kwargs)¶
__init__(websiteURL: str, websiteKey: str, userAgent: str, proxyType: str, proxyAddress: str, proxyPort: str, *args, **kwargs)¶

The class is used to work with CaptchaFox.

Parameters:

  • rucaptcha_key – User API key

  • websiteURL (str) – Full URL of the captcha page

  • websiteKey (str) – The value of the key parameter. It can be found in the page source code or captured in network requests during page loading.

  • userAgent (str) – User-Agent of your browser will be used to load the captcha. Use only modern browser’s User-Agents

  • proxyType (str) – Proxy type - http, socks4, socks5

  • proxyAddress (str) – Proxy IP address or hostname

  • proxyPort (str) – Proxy port

  • method – Captcha type

  • kwargs – Not required params for task creation request
